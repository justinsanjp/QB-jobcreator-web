QB-Core Job Editor/Studio
Web-based tool for creating and editing QB-Core jobs with AI-powered assistance.

Live Demo: https://qb-jobcreator.justinsanjp.de/

Features
Visual job configuration editor

Real-time Lua code preview

AI Editor (v1.1.0): Generate/edit jobs with multiple AI providers

Export ready-to-use QB-Core job files
